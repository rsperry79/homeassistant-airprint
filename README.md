# AirPrint Server

This addon enables AirPrint functionality on your Home Assistant system, allowing Apple devices (iOS, macOS) to discover and print directly to printers connected to your network. It runs a CUPS (Common Unix Printing System) server with Avahi for Bonjour service discovery, providing a web interface for printer management and supporting SSL/TLS encryption.

## Features

- **AirPrint Support**: Seamlessly print from iOS and macOS devices without additional drivers
- **Bonjour Discovery**: Automatic printer discovery using Avahi (Bonjour) service discovery
- **CUPS Web Interface**: Full-featured web-based printer management interface
- **SSL/TLS Encryption**: Secure printing with configurable encryption options
- **User Authentication**: Support for multiple users with different permission levels
- **Printer Driver Support**: Extensive support for various printer drivers and PPD files
- **Network Printing**: Print over the network to any connected printer
- **Logging and Monitoring**: Comprehensive logging options for troubleshooting
